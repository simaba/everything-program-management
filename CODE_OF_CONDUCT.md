# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

Be respectful. Critique work, not people. Assume good faith. If someone's behavior makes the project worse for others, it gets addressed — privately first, publicly only if necessary.

## Reporting

Email **bagherisima@gmail.com** with details. Reports are kept confidential.
