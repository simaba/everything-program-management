# Everything Program Management v0.1.0

**Publication status:** Draft release notes. Confirm the public-release checklist before publishing a GitHub Release.

## Highlights

- Reusable templates for core program-management artifacts.
- Agent, skill, and command guidance for recurring PM workflows.
- Fictional examples that demonstrate artifact structure without exposing real delivery work.
- `pmkit` validation support for RAID rows and decision-memo frontmatter.
- Automated checks for the utility layer and bundled examples.

## Intended use

Use this as a structured artifact-generation and validation layer alongside established systems of record such as Jira, Asana, Excel, Confluence, or equivalent tools.

## Important limitations

- This release does not manage project execution or replace an organization's decision rights, governance process, or recordkeeping.
- Templates and examples are practitioner aids, not legal, HR, procurement, financial, or compliance advice.
- Public examples must remain fictional or fully sanitized.
